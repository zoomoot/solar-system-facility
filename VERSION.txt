════════════════════════════════════════════════════════════════════════════════
COMPONENTS
════════════════════════════════════════════════════════════════════════════════

1. Flask Backend API (app.py)
   - Port: 5050
   - Purpose: Data integration layer
   - APIs: JPL SBDB, SsODNet, Wikipedia
   - Caching: 24-hour file-based cache
   
2. Streamlit Frontend (streamlit_app.py)
   - Port: 8501
   - Purpose: Interactive data science interface
   - Features: 3D visualizations, Plotly charts, advanced filtering
   
3. Database Loader System (loader_app.py + loader_dashboard.py)
   - Ports: 5060 (API), 8601 (Dashboard)
   - Purpose: Bulk data loading into MariaDB
   - Status: Optional (not required for Streamlit app)

STANDALONE USAGE
════════════════════════════════════════════════════════════════════════════════

Quick Start:
   ./START_EXPLORER.sh

Manual Start:
   1. source venv/bin/activate
   2. python app.py &
   3. streamlit run streamlit_app.py

Direct Access:
   - Streamlit UI: http://localhost:8501
   - Flask API: http://localhost:5050
